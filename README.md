# systemfonts

<!-- badges: start -->

<!-- badges: end -->

systemfonts is a package that locates installed fonts. It uses the
system-native libraries, meaning CoreText for macOS, DirectText for
Windows, and FontConfig for Linux.

THIS IS A WORK IN PROGRESS
